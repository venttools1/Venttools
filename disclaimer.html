Technical Disclaimer

Always verify before construction

VentTools simplifies calculations and selected information from published documents. It does not replace the full installation manual, classification, Declaration of Performance, project fire strategy, approved drawing or competent-person review.

Manufacturer revisions

Manufacturers may revise products, opening tables, tested constructions and installation instructions. Confirm the document revision shown by VentTools against the latest official manufacturer publication.

Input responsibility

The user is responsible for selecting the correct manufacturer, product, installation method, supporting construction, sealing system and dimensions. An accurate calculation based on the wrong method is still an incorrect result.

Report a concern

Stop and verify the installation if a result appears inconsistent. Report suspected errors to info@venttools.com with the product, method, dimensions and manual reference.
